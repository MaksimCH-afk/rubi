### Отчёт о найденных багах и проблемах

#### Критические баги
1. **[BUG-001]** В `script.js:186` используется `console.error()` - должен быть удалён или заменён на обработку ошибок для production
2. **[BUG-002]** Отсутствуют атрибуты `width` и `height` у изображений - приводит к layout shift при загрузке
3. **[BUG-003]** В `privacy.html` отсутствует скрипт Swiper, но он не нужен для этой страницы - можно убрать лишний код или добавить проверку

#### Средние баги

5. **[BUG-005]** Нет обработки ошибок при отсутствии элементов DOM в некоторых местах (например, `phoneInput`, `emailInput` могут быть null)
6. **[BUG-006]** В функции `formatPhone` есть потенциальная проблема: если пользователь удалит все символы, поле станет пустым, а не "+7"

#### Мелкие баги и улучшения
7. **[BUG-007]** Отсутствует плавная прокрутка к якорям при клике на навигационные ссылки
8. **[BUG-008]** Нет обработки случая, когда Swiper не загрузился (CDN недоступен)
9. **[BUG-009]** В HTML есть опечатка: "About As" вместо "About Us" (строка 29 в index.html и строка 28 в privacy.html)
